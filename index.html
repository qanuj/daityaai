Why they feel special

Pure web, done right

Single-file HTML
Every demo is one .html file. Open it locally, host it anywhere, or push to GitHub Pages. No build pipeline, no node_modules, no complexity.
Instant GitHub Pages
Just enable Pages on the main branch. Your demos are live at a beautiful URL in under a minute. Works perfectly with .nojekyll.
Real algorithms & UX
Minimax game theory, state machines, keyboard handlers, localStorage persistence, smooth animations, and accessible controls.
Production polish
Responsive, touch-friendly, keyboard-first, nice micro-interactions, copy/paste support, and thoughtful empty states.
Made to be studied
Clean, well-commented code. Great for learning vanilla JS patterns, game AI, UI architecture, and modern CSS.
Timeless & portable
No frameworks that will break. These will render beautifully in any browser from 2018 to 2035+. Truly future-proof demos.

Get started in seconds

git clone https://github.com/qanuj/learning-grok.git
cd learning-grok
# Open index.html in your browser
# Or enable GitHub Pages (main branch / root)
Every demo works when you just open the file. No server, no install, no drama.
